## 🔍 Image Search App


## 📌 Project Overview

Image Search App is a web application that allows users to search for images using an external API.

Users can enter a keyword and receive a gallery of images related to the search query. The application dynamically loads results, displays them in a responsive gallery, and allows users to view images in a modal window.

The project demonstrates modern JavaScript practices, asynchronous requests, modular architecture, and working with REST APIs.

Image search applications typically interact with external APIs (for example Pixabay) to fetch image data, including previews, tags, and full-size images.
